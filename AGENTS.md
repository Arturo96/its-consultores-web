# Proyecto: Portal Web IT & Security Consultores

## Objetivo
Sitio web corporativo enfocado en generación de leads B2B en Colombia.

## Stack
- Astro.js
- Tailwind CSS
- Sitio 100% estático (sin backend)

## Reglas importantes
- No usar base de datos
- No autenticación
- No formularios (solo WhatsApp)
- Código simple, mantenible y modular

## Estructura
- pages/: páginas del sitio
- components/: componentes reutilizables
- data/: contenido estático (servicios, casos, partners)

## UX / Negocio
- El objetivo es generar leads
- Siempre incluir CTA a WhatsApp
- Lenguaje corporativo y profesional
- Enfocado en empresas (B2B)

## Convenciones
- Componentes reutilizables
- Código limpio y legible
- No sobreingeniería
